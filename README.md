# PIDAN-journal

![](PIDAN-journal.png)

Sometimes I read some posts and wanted to keep a copy locally. So I archived and view them with [Retype](https://retype.com/).

## Tools

- MarkDownload - Markdown Web Clipper (Browser Plugin)
- QOwnNotes
- YOGA Image Optimizer
- waifu2x - Snowshell
- ScreenToGif
